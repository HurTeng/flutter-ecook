## 项目结构
***
<pre>
├── android            // android相关目录
├── ios                // ios相关目录
├── lib                // flutter目录
│   ├── common         // 通用类
│   ├── list           // 列表组件
│   ├── page           // 内容页面 
│   ├── widget         // 封装的控件
│   └── main.dart      // flutter入口文件
└── pubspec.yaml       // 项目配置文件
</pre>

## 项目安装
``` bash
# 获取项目
git clone https://github.com/HurTeng/flutter-ecook.git

# 进入文件夹
cd flutter-ecook

# 安装依赖
flutter packages get

# 运行项目
flutter run
```
